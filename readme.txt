This plugin allows you to disply the link to a bbPress member's profile using the shortcode [bbp_profile_link_shortcode]. That's it!

== Frequently Asked Questions ==

**Can I put this in a menu/in a post/on a page/etc.?**
Yep! You can use it anywhere shortcodes are executed.

**What happens if the user is logged out?**
It breaks! You'll need to use another plugin to manage logged in/logged out context.

**Can this plugin do ANYTHING besides print the user's profile link?**
No!	Or, not yet at least.
